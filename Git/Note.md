# Tìm hiểu Git

## Git command

- Clone project: git clone
- Add to commit: git add
- Commit: git commit
- Push: git push
- Update: git pull
- Reset : git reset
- Tách branch: git checkout -b
- Chuyển branch: git checkout 
- Merge branch: git merge
- Diff check: git diff
- Status check: git status
- Lưu thay đổi vào local: git stash"
